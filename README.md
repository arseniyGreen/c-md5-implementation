### Installation on Linux

1. Clone the repo
   ```sh
   git clone https://github.com/arseniyGreen/c-md5-implementation
   ```
2. Compile with g++
   ```sh
   cd c-md5-implementation
   g++ md5.cpp && ./a.out
   ```
